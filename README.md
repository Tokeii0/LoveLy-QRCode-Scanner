# LoveLy-QRCode-Scanner
### 基于 WeChatCV/opencv_3rdparty 的Python解码脚本
调用了WeChatCV/opencv_3rdparty的解码模型,然后对图片进行，亮度、对比度、模糊度调整，爆破解码

对AI生成的二维码有奇效
